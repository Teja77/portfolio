Inspiration

Ever since I was a kid ,I have had a passion for technology and how it changes our lives. I still remember using the first mobile device (Nokia) to play games, text friends for the first time. It felt magical to see all the things we could do in a small portable device. As I have journeyed through my career my passion has only grown stronger. With so many advancements and technologies to adapt to my passion had led me to studying Virtual reality and Software development.

My experience in Virtual reality ranges from using Unity 3d, shaders using shadergraph, particle effects, optimising applications for mobile devices, 3d modelling, texture mapping and animations.
